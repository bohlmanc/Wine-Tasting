The improvement backlog lives in **CLAUDE-TODO.md** at the project root. It is the single source of truth for task status (`not started` | `pending` | `complete`).

Before starting any task: read CLAUDE-TODO.md, confirm current status, and mark the item `pending` before coding. Mark `complete` when done.
